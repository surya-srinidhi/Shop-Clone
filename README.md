# 🛒 Amazon Clone

A front-end clone of the Amazon homepage built to practice modern web development skills. This project replicates the core layout and styling of the famous e-commerce site.

## 🚀 Features

- **Iconic Navigation Bar:** Accurately styled top navigation featuring logo, address selector, search bar with dropdown, and cart.
- **Product Grid:** Responsive flex/grid layout showcasing different categories of items similar to the real Amazon page.
- **Footer Structure:** Multi-section footer including 'Back to Top' functionality, and detailed links.
- **Basic Interactivity:** Includes JavaScript functionality such as a smooth-scrolling "Back to Top" button for an improved user experience.

##  Technologies Used

- **HTML5:** Semantic HTML layout and structure.
- **CSS3:** Custom styling, Flexbox layouts, background images, and hover effects.
- **JavaScript (Vanilla):** DOM manipulation and event listeners.
- **FontAwesome:** Scalable vector icons for the search bar, cart, and location pin.

##  Project Structure

- `index.html` - The main rendering document.
- `style.css` - All custom styling rules and visual layouts.
- `script.js` - Interactive elements.
- Image assets (`box*_image.png`, `hero_image.png`, `amazon_logo.png`) - Visual placeholders mirroring the real site.

##  Getting Started

To view this project locally, there's no server or build process required:

1. Clone or download this repository.
2. Ensure you have the `index.html`, `style.css`, `script.js`, and all the `.png` images in the same folder.
3. Double-click `index.html` to open it in your default web browser.

> **Note:** The project was built primarily with desktop displays in mind. Mobile responsiveness may be limited as the main focus was on structural layout and Flexbox practice.

## 📝 Learning Outcomes

This project was built to reinforce core web design concepts including:
- Utilizing CSS Flexbox for complex, real-world layout structures.
- Practical application of CSS pseudo-classes for interactive hover states.
- Clean integration of third-party icon libraries (Font Awesome).
- Basic JavaScript event handling.
